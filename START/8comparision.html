The algorithm to compare two strings is simple:
Compare first characters of both strings.
If the first one is greater(or less), then the first string is greater(or less) than the second. We’re done. Otherwise if first characters are equal, compare the second characters the same way. If both strings ended simultaneously, then they are equal. Otherwise the longer string is greater.
